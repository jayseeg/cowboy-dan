**Requirements**

* node @7.5
* yarn

**Setup**

create `server/.env`
with these contents:
```
WALLET_GUID = yourBlockchainWalletGUID
WALLET_PASSWORD = yourBlockchainMerchantPassword
WALLET_SECOND_PASSWORD = yourBlockchainMerchantSecondPassword
```

```
yarn start
```

that should start a few servers

navigate to localhost:3001
